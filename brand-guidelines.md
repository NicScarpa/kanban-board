# Brand Guidelines

Questo file definisce i font e i colori da usare nel progetto per mantenere coerenza con il brand.

## Colori

### Colori Principali
```css
--color-primary: #3B82F6;     /* Blue 500 */
--color-secondary: #6366F1;   /* Indigo 500 */
--color-accent: #8B5CF6;      /* Violet 500 */
```

### Colori Neutri
```css
--color-background: #0F172A;  /* Slate 900 */
--color-surface: #1E293B;     /* Slate 800 */
--color-text: #F8FAFC;        /* Slate 50 */
--color-text-muted: #94A3B8;  /* Slate 400 */
```

### Colori di Stato
```css
--color-success: #22C55E;     /* Green 500 */
--color-warning: #F59E0B;     /* Amber 500 */
--color-error: #EF4444;       /* Red 500 */
--color-info: #0EA5E9;        /* Sky 500 */
```

## Font

### Font Principale (Titoli)
- **Font**: Inter
- **Weights**: 400, 500, 600, 700
- **Link**: `https://fonts.google.com/specimen/Inter`

### Font Secondario (Body)
- **Font**: Inter
- **Weights**: 400, 500
- **Link**: `https://fonts.google.com/specimen/Inter`

### Font Monospace (Codice)
- **Font**: JetBrains Mono
- **Weights**: 400, 500
- **Link**: `https://fonts.google.com/specimen/JetBrains+Mono`

## Tipografia

### Scale
```css
--text-xs: 0.75rem;    /* 12px */
--text-sm: 0.875rem;   /* 14px */
--text-base: 1rem;     /* 16px */
--text-lg: 1.125rem;   /* 18px */
--text-xl: 1.25rem;    /* 20px */
--text-2xl: 1.5rem;    /* 24px */
--text-3xl: 1.875rem;  /* 30px */
--text-4xl: 2.25rem;   /* 36px */
```

## Spaziatura

```css
--spacing-1: 0.25rem;  /* 4px */
--spacing-2: 0.5rem;   /* 8px */
--spacing-3: 0.75rem;  /* 12px */
--spacing-4: 1rem;     /* 16px */
--spacing-6: 1.5rem;   /* 24px */
--spacing-8: 2rem;     /* 32px */
--spacing-12: 3rem;    /* 48px */
--spacing-16: 4rem;    /* 64px */
```

## Border Radius

```css
--radius-sm: 0.25rem;  /* 4px */
--radius-md: 0.5rem;   /* 8px */
--radius-lg: 0.75rem;  /* 12px */
--radius-xl: 1rem;     /* 16px */
--radius-full: 9999px;
```

---

> **Nota**: Modifica questi valori in base alle esigenze del tuo brand.
